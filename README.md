# E-Commerce Data Warehousing Project (Brazilian Olist Dataset)

## Project Overview
This project focuses on building a SQL-based data warehousing solution using the Brazilian Olist E-commerce dataset.
The project includes data cleaning, transformation from Bronze to Silver layer, data quality checks, and business insights generation.

## Tools Used
- MySQL
- SQL
- Microsoft Word
- Kaggle Dataset (Brazilian Olist)

## Dataset Tables Used
- Orders
- Order Items
- Products

## Key Steps Performed
1. Data Import (Bronze Layer)
2. Data Cleaning (Silver Layer)
3. Datatype Conversion
4. Null Value Handling
5. Duplicate Removal
6. Invalid Date Correction
7. Foreign Key Validation
8. Business Insight Queries

## Sample Business Questions Answered
- Total Orders
- Top Selling Products
- Revenue Analysis
- Delivery Performance

## Project Outcome
Built a structured SQL data warehouse with cleansed and validated data for business analysis.
